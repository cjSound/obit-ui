# obit-ui

## Usage
1. Install package [obit-ui]
   ```sh
   npm i -save-dev obit-ui
   ```
2. Get Started
   ```vue
    <template>
    <div id="app">
        <svg-icon icon-class="location" />
    </div>
    </template>
    <script>
    import { svgIcon } from 'obit-ui'
    import 'obit-ui/dist/obit.css'
    export default {
        name: 'app',
        components: {
            svgIcon
        }
    }
    </script>

   ```
